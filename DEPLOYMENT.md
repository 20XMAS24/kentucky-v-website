# Кук деплойтить и кастомизировать сайт

## Аддинг Логотипа

### Метод 1: Локально

1. Составите ваш логотип PNG файл с прозрачным фоном
2. Назовите его `logo.png`
3. Местим в папке `assets/`

### Метод 2: Онлайн URL

Если вы хотите использовать URL вместо локального файла:

Оторедактируйте строку в `index.html`:

```html
<img src="https://your-image-url.com/logo.png" alt="Kentucky V Logo" class="logo-image">
```

## Варианты распределения

### На GitHub Pages

1. Го в Settings навес и данная репозитория
2. На Видит при Пагес и выбрать `main` ветку
3. Ваш сайт будет авоступны на: `https://20XMAS24.github.io/kentucky-v-website`

### По собственному домену

1. Клонируйте репозиторий
2. Загрузите все файлы на ваш веб-сервер
3. Настройте точку вкода

### На Vercel/Netlify

1. Подключите ваш GitHub репозиторий
2. Сайт деплойтится автоматически при каждом пуше

## Кустомизация Стилей

### Изменение цветов

В `styles.css` найдите основные цвета:

```css
:root {
  --primary-color: #ff4444;      /* Основной красный */
  --primary-hover: #ff6666;      /* Красный с наведением */
  --primary-dark: #cc0000;       /* Немно темные не */
  --bg-color: #0a0a0a;          /* Явные светлых звуков */
  --bg-secondary: #1a1a1a;       /* Вторичные светлые */
}
```

### Пример: Используя гения индиго

Кагда используете цвет темы:

```css
--primary-color: #6366f1;       /* Индиго */
--primary-hover: #818cf8;       /* Лит ов индиго */
--primary-dark: #4f46e5;        /* Темные индиго */
```

## Год FAQ

### Ракзсы нююи сайт показывает плацехолдер лого?

Убедитесь, что `assets/logo.png` существует и имеет правильные размеры (300x300px минимум).

### Как добавить любой раздел?

Добавьте новую `<section>` в `index.html` и стили в `styles.css`.

### Сайт не работает осанде?

1. Открыть относительные подразумеваемые в девтоолс
2. Проверьте, что все все пути пофайла говорят релативным путем
3. Ю лога в консоли для ошибок
